= WALL GEN

Simple utils to generate wallpapers and images in c.

== Build
This project use [nob.h](https://github.com/tsoding/nob.h.git) -- build
system. So, to build just run.

```sh
$ cc ./nob.c -o ./nob
```

== Usage:

```sh
$ ./nob command ...args
```

== Example

```sh
$ ./nob grid --bg-color green --line-color yellow
```
